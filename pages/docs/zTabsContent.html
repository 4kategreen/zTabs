.zTabs('content'[, filter])

Returns a jQuery object representing the content of the specified tab or tabs -- the data that is displayed when the user selects the tab.

The initial selector should be a jQuery object representing one or more tabs.

This takes an optional filter argument that narrows the returned object to represent only things in the content which match the filter. The optional filter uses jQuery-style selector syntax, such as "#mainTabs" or ".classname".

Examples:
// Changes the content associated with the "tabOne" and "blue" tabs to 'Something new!'
$('#tabOne, #blue').zTabs('content').html('Something new!');		

// Finds the current tab and changes its content to 'Loading...'
$('#mainTabs').zTabs('current').zTabs('content').html('Loading...');	

// Changes the text for all the paragraphs in the current tab's content.
$('#mainTabs').zTabs('current').zTabs('content', 'p').html('Put this text in all the paragraphs');

Note: Because ".zTabs('current').zTabs('content')" is a common pattern, ".zTabs('cc')" is provided as a shortcut.
