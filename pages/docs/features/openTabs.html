Opening Tabs

zTabs supports a number of ways for opening tabs.

Normal Tabs:

The first way for opening a tab is to set it up in HTML so that it is presented as a default tab when its row is loaded by zTabs.

Example:
// The HTML used to build the first tab in this Docs subrow
<ul>
	<li id='zTabsDocs'><a href='pages/docs/list.html'>zTabs Documentation</a></li>
</ul>
Dynamic Tabs:

zTabs also supports opening tabs dynamically, out of the box. HTML anchors and button elements can both be made to launch new tabs when clicked.

To cause one of these elements to open a new tab, add the zTab class to its HTML.

Example:
// Anchor with class zTab
<a href='pages/docs/features/helloworld.txt' class='zTab'>Hello World!</a>
Hello World!

// Button with class zTab
// The data-contenturl attribute specifies the content that will be loaded in the new tab
<button type='button' class='zTab' data-contenturl='pages/docs/features/helloworld.txt'>Hello World!</button>
