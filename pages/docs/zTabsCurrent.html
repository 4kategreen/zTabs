.zTabs('current')

Returns a jQuery object representing the current tab's list item.

The initial selector can be the id of any existing tab or ul in the tabset. This allows you to target the current tab specifically, without having to know any of its identifying details. This behavior is valuable when working with dynamically created tabs.

Example:
// Changes the label of the current tab
$('#mainTabs').zTabs('current').zTabs('property', {label: 'New Label'});	

// Returns the label of the current tab
// The initial selector can be any valid tab or ul within the tabset for the current tab to be returned
$('#demos').zTabs('current').zTabs('property', 'label');

Note: Because ".zTabs('current').zTabs('content')" is a common pattern, ".zTabs('cc')" is provided as a shortcut.
