.zTabs('parentContent'[, <filter>])

Returns a jQuery object representing the first content area in the DOM which contains a match for the initial selector.

The initial selector can be anything which can appear in the content area, however .zTabs('parentContent') will only match elements which currently exist in the DOM.

This takes an optional filter argument that narrows the returned object to represent only things in the content which match the filter. The optional filter uses jQuery-style selector syntax, such as "#mainTabs" or ".classname".

This function is similar to .zTabs('cc'), in that you can use it to target a specific content area without knowing any of the parent tab's identifying details. This behavior is valuable if tabs have been generated dynamically, and when a content element is a more meaningful identifier than whatever tab contains it.

Examples:
// Retrieves the current user's username
// Populates the "firstname" field of the "testForm" form
var username = $('div.username').html();
$('#testForm').zTabs('parentContent', 'input[name="firstname"]').val(username);
