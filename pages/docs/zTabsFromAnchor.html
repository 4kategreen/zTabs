.zTabs('fromAnchor', <options>) Turn an anchor tag into a link that spawns a new tab. This is a convenient way to deal with a list or table of links. The id of the new tab can be set my adding a tabId attribute to the anchor tag.

Please note that as of zTabs 1.0.6, if you're also using jQuery 1.3 or higher, setting the class of an anchor tag to "zTab" will make it open as a new tab regardless of when that anchor tab was introduced to the DOM. That may mean that you don't ever need the zTabsFromAnchor class. If you want to change that class or make all anchors open a new tab, set the property fromAnchor to the selector of your choice when you initialize zTabs.

Add a tab to a set of tabs. This can be called on a unordered list (<ul>) or an list item (<li>). If called on an <li> it will show the <li> and add the new tab to the sub-tabs of that <li>. The sub-tabs of that <li> need not exist when zTabsAdd is called as long as they will exist after the <li> has been shown.
Options:
tabid - The id that should be set in the list item. Example: <a tabId='foobar'>
closeable - Can the user close the tab. (true, false) The default is true (it used to be false).
onwake - A function to call every time the tab is shown.
onsleep - A function that gets called every time a different tab is pressed and this is a selected tab. If the onSleep function returns false then the switch to the new tab is cancelled.
oncreate - A function that gets called only the first time a tab is shown.
onclose - A function to be called if the user presses the close button. If this function returns false then the close is cancelled.
cache - Determines if the tab should go to the server and load contentURL every time the tab is clicked (false) or just the first time (true). The default is true.
Examples:
		// add a new tab as a sub-tab of tabThree
		$('#users').find('a').zTabs('fromAnchor');
		
		<table id='users'>
			<tr><th>Name</th><th>Shoe Size</th></tr>
			<tr>
				<td><a href='foo/daisy.html'>Daisy Roberson</a></td><td>7</td>
			</tr>
			<tr>
				<td><a href='foo/maxwell.html'>Maxwell Roberson</a></td><td>3</td>
			</tr>
		</table>
