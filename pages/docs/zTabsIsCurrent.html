.zTabs('isCurrent')

Returns true if the given tab is the current tab, and false otherwise.

The initial selector should be a jQuery object representing a single tab. If more than one tab is in the jQuery set, then only the first tab in the set is used.

Example:
// Returns true if tabOne is the current tab
$('#tabOne').zTabs('isCurrent');	

// Closes every tab in the second row that isn't the current tab
$('ul.row2 li').each(function(){ 
	if(!$(this).zTabs('isCurrent')) { 
		$(this).zTabs('close'); 
	} 
});
