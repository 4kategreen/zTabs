.zTabs('property', <name>)

Returns the value of the named property for the given tab.

The initial selector should be a jQuery object representing a single tab. If more than one tab is in the jQuery set, then only the first tab in the set is used.

Examples:
// Returns the label of the tab
$("#tabOne").zTabs('property', 'label');	

// Executes the onclose function for the zTabsProperty tab.
$('#zTabsProperty').zTabs('property', 'onclose')();
.zTabs('property', <name>, <value>)

Sets the named property of the given tab, defined by key, to the value.

If more than one tab is given by the initial selector, the named property will be set for each of the tabs.

Returns a jQuery object representing the tabs whose properties have been changed.

Examples:
// Changes the label of the current tab
$('#mainTabs').zTabs('current').zTabs('property', 'label', 'New Label');

// Adds an onclose function to the zTabsProperty tab which alerts "hello world!"
$('#zTabsProperty').zTabs('property', 'onclose', function(){ alert("hello world!"); });
.zTabs('property', <properties>)

Sets the values of the named properties passed in as an object (map/associative array).

If more than one tab is given by the initial selector, the named property will be set for each of the tabs.

Returns a jQuery object representing the tabs whose properties have been changed.

Examples:
// Changes the label of the tab
$("#tabOne").zTabs('property', {label:'Tab'});	

// Sets the tab to closeable so the close "x" is displayed
$("#tabOne").zTabs('property', {closeable:true});		

// Sets the tab to closeable, but also adds an onclose function which prevents the tab from closing
// The label is updated to reflect the changed properties
$("#tabOne").zTabsProperty({
	closeable: true,
	onclose: function() {
		alert("No, you may not close this tab");
		return false;
	},
	label: "Can't Close This"
});
