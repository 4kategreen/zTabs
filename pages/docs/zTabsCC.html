.zTabs('cc'[, <filter>])

Returns a jQuery object representing the content of the current tab -- the data that is displayed when the user selects the tab.

The initial selector can be the id of any existing tab or ul in the tabset. This allows you to target the content of the current tab specifically, without having to know any of its exact details. This behavior is valuable when working with dynamically created tabs.

This takes an optional filter argument that narrows the returned object to represent only things in the content which match the filter. The optional filter uses jQuery-style selector syntax, such as "#mainTabs" or ".classname".

Note: This is a convenience method, provided as a shortcut for the common ".zTabs('current').zTabs('content')" pattern.

Example:
// Changes all the text in pre tags in the current content area to blue.
$('#mainTabs').zTabs('cc', 'pre').css({color:'blue'});
