zTabs

Welcome to zTabs. This jQuery plug-in creates a hierarchical set of lazy-loading tabs from an initial unordered list of anchor tags. We've found it a useful interface for web applications. We hope you find it valuable as well. It will probably be set up with the MIT-style license in the near future.

This package is contains documentation and examples. It's presented in a zTabs based UI so you'll find it a useful test-bed for experimenting with zTabs.

Quick Hints

Because this anchor tag has a class of "zTab" it automatically opens a new tab: Anchor with zTab Class

The zGrayScreen module gives you an easy way to do dialog boxes: zGrayScreen. It's similar to the jQuery plug-in called lightbox but has some benefits like a faster loading time, better support for long content and support for text only messages.

$().zGrayScreen({title:'foobar', html:'<p>Hello World</p>', size:'small'});

This is an example of a text only message: zGrayScreen type:'text'

$().zGrayScreen({title:'foobar', html:'<p>Hello World</p>', size:'small', type:'text'});

Build into this web application template is a simple way to send your user a notice or status update: messageToUser. To use this, just call messageToUser and pass it the HTML you want to display. Documentation will be available soon on Spork, it it's not already.

messageToUser('Hello World');

zTabs will keep track of open tabs in the browser via localStorage. If you need to clear that out use this zTabs method: $('#mainTabs').zTabs('clear');
